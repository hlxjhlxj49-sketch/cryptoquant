# 🪙 加密货币量化交易系统

> 一站式数据抓取 · 策略回测 · 量化交易平台

**适合零基础用户**，所有操作通过可视化网页界面完成，无需命令行。

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 📊 **数据管理** | 一键下载币安/OKX/Bybit等交易所K线数据，本地SQLite存储 |
| 📈 **行情看板** | 交互式K线图 + MA均线 + KDJ + 成交量，支持缩放拖拽 |
| 🧠 **策略工坊** | 大白话描述交易想法，自动生成策略代码！ |
| ⏮️ **策略回测** | 历史数据验证策略，夏普比率/最大回撤/胜率一目了然 |
| 💹 **模拟交易** | 虚拟资金 + 实时行情，零风险演练 |
| 🚀 **实盘交易** | 连接真实交易所API，一键执行交易 |

## 🚀 快速开始

### 1. 安装 Python

如果还没安装 Python：
- 下载地址：https://www.python.org/downloads/
- 安装时 **勾选 "Add Python to PATH"**
- 版本要求：Python 3.9+

### 2. 启动系统

```
双击 E:\crypto_quant\启动.bat
```

系统会自动：
1. 检查 Python 环境
2. 安装依赖包（首次运行约需1-2分钟）
3. 打开浏览器，显示系统界面

### 3. 开始使用

1. **下载数据** → 进入「📊 数据管理」，选择币安 → BTC/USDT → 1小时 → 下载
2. **查看行情** → 进入「📈 行情看板」，选择BTC/USDT，点击加载
3. **创建策略** → 进入「🧠 策略工坊」，输入策略描述，生成代码
4. **回测验证** → 进入「⏮️ 策略回测」，选策略→跑回测→看结果
5. **模拟演练** → 进入「💹 模拟交易」，用虚拟资金实战

## 📖 策略示例

### 因子构建器 — 用自然语言写策略

在「策略工坊」中输入以下描述，系统自动生成策略代码：

**示例1：双均线交叉**
```
当5日均线上穿20日均线时买入，下穿时卖出
```

**示例2：KDJ超买超卖**
```
当KDJ的K值低于20时买入，高于80时卖出
```

**示例3：组合策略**
```
MACD金叉且KDJ在50以下金叉时买入，任一死叉时卖出
```

**示例4：放量突破**
```
价格突破20日最高价且成交量大于1.5倍均量时买入
```

## ⚙️ 配置说明

编辑 `config/settings.yaml` 可以修改：

- **默认交易所**：`exchange.default: "binance"`
- **默认K线周期**：`fetcher.default_timeframe: "1h"`
- **交易模式**：`trading.mode: "spot"` 现货 / `"futures"` 合约
- **风险管理**：止损止盈比例、仓位限制等

## ❓ 常见问题

**Q: 数据下载很慢？**
A: 受交易所API限速影响，大量下载时较慢。建议先下载1个币种测试。可以在 `settings.yaml` 中调整 `rate_limit_pause`。

**Q: 提示 "ccxt.BadSymbol" 错误？**
A: 交易对名称需要严格匹配。币安现货是 `BTC/USDT`，合约是 `BTC/USDT:USDT`。

**Q: 回测结果看起来太好？**
A: 可能因为数据量不够（至少需要100条K线），或者策略存在未来函数。请增加回测数据量检验。

**Q: 如何添加新交易所？**
A: CCXT 支持100+交易所，在「数据管理」页面选择即可。需要API Key的交易所请先注册。

**Q: 实盘交易如何配置？**
A: 在交易所创建API Key（只开交易权限，不开提现权限），填入「实盘交易」页面。
